# Learn How to Program Clone

## Specs

* Lessons Model
  * Name
  * Text Content
  * Number
  * CRUD Functionality
  * Belongs to Section
* Next Lesson link goes to next lesson in order (Based on lesson #)
* Previous Lesson link goes to previous lesson in order
* Section Model
  * Has many lessons
* Table of contents for Section shows all Lessons
  * Ordered by lesson #
* Course
  * Has many sections
